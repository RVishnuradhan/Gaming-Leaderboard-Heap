# Gaming Leaderboard using Max Heap

## Description
This project implements a live gaming leaderboard using a Max Heap (Binary Tree). The player with the highest score is always maintained at the top for instant access.

## Features
- Add player with score
- View top player instantly
- Display leaderboard
- Remove top player

## How it works
A Max Heap is used where the highest score is always stored at the root node. This ensures that the top player can be accessed in constant time.

## Time Complexity
- Insert Player: O(log n)
- Remove Top Player: O(log n)
- Get Top Player: O(1)

## Concepts Used
- Heap (Binary Tree)
- Priority Queue
- Arrays in C
- Data Structures

## Example
Players:
A - 100  
B - 200  
C - 150  

Top Player: B
